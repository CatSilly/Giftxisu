[Giftxisu](https://catsilly.github.io/Giftxisu/)
===
![LOGO](res/logo.png)
A simple "gift-catching" game.
Built using HTML, CSS, and JS.

Featured
--------
- You are the person pulling a basket to catch gifts; each time you catch a gift, you get 1 point. 
- There are some trap gifts (Poison Potion, Red Thorns, etc.) that will make you lose your heart. 
- The game will get faster and faster over time.

Bug Report
----------
Encountering errors? Report them immediately in *Issues* so we can fix them.
